# 綱領

リバタリアン社会主義直接行動は、民衆の自発的な直接行動によって社会を変革する運動体である。我々が掲げる理念とは、支配と疎外から自らを解放した自由な諸個人による自発的な連帯に基づく社会の創造、これが即ち[リバタリアン社会主義](https://files.libcom.org/files/I1.pdf)と呼ばれる思想である。

リバタリアン社会主義の基本原則は、簡単に纏めれば「自治」という理想に集約できる。自治とは、自らに関する事柄を自らの意思によって決定し、行為する事である。我々は、自治は人間の生活における最も基本的な原理であり、社会の全ての領域を自治によって運営しなければならないと考えている。自分の事は自分で決める、当たり前の事だろう？

だが、こんな当たり前の、根源的な自己決定の欲望すらも、未だ実現できていない。国家や資本主義などの抑圧構造が存在し、人々を支配し疎外しているからである。自治を成り立たせる上での前提条件が、外部の権威や支配を完全に排除する事である。我々は、これらの抑圧構造を根本から打倒・廃絶する為に闘わなければならない。

## 歴史の総括

社会主義は大きく二つに分類出来る。リバタリアン社会主義と、我々と対比される他の潮流。ここでは「権威主義的社会主義」と呼ぼう。権威主義的社会主義とは、労働者や民衆ではなく、国家や党の権威によって社会主義を実現するという潮流である。これにはボルシェヴィズムやスターリン主義、社会民主主義などが含まれる。一方の我々には、アナキストや革命的サンディカリストから、相互主義者、アソシエーショニスト、リバタリアン・マルクス主義者、オートノミスト、評議会共産主義者、などが含まれる。

国家による社会主義が矛盾していることは、マルクスが国家を階級的抑圧装置と規定した事を鑑みれば直ぐに理解できるだろう。

ソ連スターリニズムは官僚主義によって国家資本主義・司令型計画経済に堕落した挙句、崩壊した。社会民主主義も、穏健な議会路線を通じた再分配や規制による資本主義の修正を指向するなど、国家の政策にほとんど依存してしまっている。

これらの権威主義的社会主義はどちらにせよ、資本主義的生産様式とその階級的抑圧機構である国家を廃棄することはなく、労働者を疎外から解放する事はなかった。果たして、労働者がブルジョワ的支配から解放されるのに、権威や統制は必要だろうか？必要な所か、自発的な連帯を妨げ、むしろ有害でしかない。よって、真の社会主義は、リバタリアン的でなければならない。さもなくば労働者が自己解放を成し遂げることは無い。

## 運動

直接行動とは、議会選挙など「正規の」間接的・制度的な政治手続きを経ずに、民衆自らの主体的・直接的な「力」の発揮により社会に働き掛ける政治、そして革命実践である。リバタリアン社会主義者の多くは、国家制度の中で政治をする事を拒むので、我々の理念と直接行動の戦術は非常に相性が良いのである。

我々が出来ることの例を以下に挙げるが、これだけに限定する事は無い。デモ、街宣、カウンター街宣、ビラ配り、ポスター貼り、落書き、学生運動、全学バリケード封鎖、労働運動、ストライキ、座り込み、炊き出し、困窮者支援、一時的自律ゾーン、スクワッター、状況の創出、などなど…

**何より我々は、自発的そして遊戯的な革命実践を是とする**

逆に言えば、自発性に依拠しない活動は否定される。民衆を「代表」に任せて政治から切り離す一切の権威的手法、即ち間接行動は、反革命行為であり許されない。あくまで直接行動を至上のものとする。民衆は常に自らに関わる事、政治を自らの意志によって決定しなければならないのだ。

また、我々が目指すべき新しい社会の仕組み（生産関係や交換様式、技術的インフラなど）を考察し、[予示的政治](https://ja.theanarchistlibrary.org/library/yukihikotakahashi)（Prefigurative Politics）として社会実装に移さねばならない。その社会システムとしての是非、実用的なメリット／デメリットから、リバタリアン的原則に一致してるかを吟味し試行錯誤しながら、国家と資本主義を廃棄した後に開かれる地平、リバタリアン社会主義の建設を実際に進める必要がある。


## 組織

我々の組織そのものが、リバタリアン社会主義を体現する必要がある。構成員が組織に属する事によって強制や支配を受ける事は一切あってはならない。意志決定はリバタリアン的原理に則り、関係者各人の自由意志による合意を基本とする。

この綱領では制度を詳細には規定しない。人々が集まるに連れて自己組織化しながら、制度は絶えずその場その場の合意によって生成されて続けるからだ。

我々に加盟する者に求められる条件は、この綱領と以下の原則への同意である。

1. 自由意志と自発性の擁護
2. 支配と権威の根本的否定
3. 国家と資本主義の廃棄

このリバタリアン的原則に依拠した上で、君が望む様に好きな行動を提起し、自発的な参加者を集める事ができる。何でも出来る。可能性は無限大だ。何故なら、誰も偉そうに君に命令してくる指導部は居ないのだから！

_万国の労働者よ、再結集せよ！_

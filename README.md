# リバタリアン社会主義直接行動 綱領

リバタリアン社会主義直接行動は、間接的な議会政治に頼らず民衆の自発的な直接行動によって急進的に社会を変革する運動体である。我々が目指す理念とは、支配と疎外から自らを解放した自由な諸個人による自発的な連帯に基づく社会の創造、これが即ち[リバタリアン社会主義](https://files.libcom.org/files/I1.pdf)と呼ばれる思想である。

ここで言うリバタリアン（Libertarian）とは、英語辞典を開けば分かるように、権威主義（Authoritarian）の対義語に位置付けられ、自由意志や自発性を尊重するという意味を持つ幅広く使われる言葉である。世論でリバタリアンが右派的な市場原理主義と関連付けられるのは、表面的な理解に過ぎない。アメリカ合衆国以外、特に欧州大陸では、辞書通りに

また、社会主義の理解も正す必要がある。社会主義の定義について偏らずに言うと、貨幣の物象化／物神崇拝（フェティシズム）や労働疎外など資本主義の諸問題の克服を目指す思想の総称と言える。だが、今日で一般的な文脈で捉えられる「社会主義」という言葉は、スターリニズムの独裁の印象か、良いイメージでも社会民主主義である事が多い。ソ連スターリニズムは官僚主義によって国家資本主義／司令型計画経済に堕落した挙句、崩壊した。社会民主主義も、穏健な議会路線を通じて再分配や規制による資本主義の修正を指向するなど、権威主義的社会主義と断じざるを得ない。

どちらにせよ、資本主義的生産様式とその階級的抑圧機構である国家を廃棄することはなく、労働者を疎外から解放する事はなかった。これらの潮流は、労働者がブルジョワ的支配から解放されるのに、権威や統制は必要だろうか？必要な所か、自発的な連帯を妨げ、むしろ有害でしかない。よって、真の社会主義は、リバタリアン的でなければならない。さもなくば労働者が自己解放を成し遂げることは無い。20世紀の歴史を通じて権威主義的社会主義の失敗はもはや目に見えている。権威主義的社会主義では無い、もう一つの社会主義の歴史を呼び覚ます必要がある。だからこそ、ここで我々はリバタリアン社会主義を掲げる必要があるのだ。

## 運動

我々は、組織の名称から理解出来る様に、直接行動を我々の活動の中心に据える。以下に我々が出来ることの例を挙げるが、これだけに限定する事は無い。デモ、街宣、カウンター街宣、ビラ配り、ポスター貼り、落書き、学生運動、全学バリケード封鎖、労働運動、ストライキ、座り込み、炊き出し、困窮者支援、一時的自律ゾーン、スクワッター、状況の創出、などなど…

**何より我々は、自発的そして遊戯的な革命実践を是とする**

逆に言えば、自発性に依拠しない活動は否定される。議会進出や交渉など「代表」を立てて民衆を政治から切り離す一切の権威的手法、即ち間接行動は、反革命行為であり許されない。あくまで直接行動を至上のものとする。民衆は常に自らに関わる事、政治を自らの自由意志によって決定しなければならないのだ。

また、我々が目指すべき新しい社会の仕組み、生産様式／交換様式を考察し、[予示的政治](https://ja.theanarchistlibrary.org/library/yukihikotakahashi)（Prefigurative Politics）として行動の実践に移し、社会関係としてのメリット／デメリットを吟味し試行錯誤しながら、国家と資本主義を廃棄した後に開かれる地平、リバタリアン社会主義の建設を実際に進める必要がある。


## 組織

我々の組織そのものが、リバタリアン社会主義を体現する必要がある。この綱領では制度を詳細に規定しない。人々が集まるに連れて、自己組織化しながら、制度が生成されて行くのだ。意志決定はリバタリアン的原則に則り、関係者各人の自由意志による合意を基本とする。我々に加盟する者に求められる条件は、この綱領と以下の原則への同意である。

1. 個人の自由意志と自発性の擁護
2. 近代の国家と資本主義の根本的否定
3. 新しい自律的な社会関係の創造

このリバタリアン的原則に依拠した上で、君が望む様に好きな行動を提起し、自発的な参加者を集める事ができる。何でも出来る。可能性は無限大だ。何故なら、誰も偉そうに君に命令してくる指導部は居ないのだから！

ここに、アナキストや革命的サンディカリストから、相互主義者、アソシエーショニスト、リバタリアン・マルクス主義者、オートノミスト、評議会共産主義者、トロツキストに至るまで、反権威主義的社会主義諸勢力の幅広い結集を呼び掛ける。

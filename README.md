# リバタリアン社会主義直接行動 綱領 

リバタリアン社会主義直接行動（以下「リバ直」）は、国家と資本主義を廃棄し近代を超克する事を目的とした運動体である。我々が目指す理念とは、支配と疎外から自己解放した自由な諸個人による自発的な社会の創造、即ちリバタリアン社会主義と呼ばれる思想である。

リバタリアン社会主義とは何か。国家と資本主義の両方に反対し、自由な個人を前提とした社会主義である。ここでいうリバタリアン(Libertarian)とは、英語圏において権威主義(Authoritarian)の対義語に位置付けられ、自由意志や自発性を尊重するという幅広い意味を持つ言葉である。世論でリバタリアンが右派的な市場原理主義と関連付けられるのは、表面的な理解、むしろ言葉の簒奪に過ぎない。

また、社会主義の理解も正す必要がある。社会主義とは偏らずに言うと貨幣の物象化／物神崇拝や労働疎外など資本主義の諸問題の克服を目指す思想の総称と言える。だが、歴史的に社会主義を標榜してきた運動の中で、国家による再分配や規制を指向する社会民主主義や、国家資本主義に歪曲したスターリニズムなどは、権威主義的社会主義と断じざるを得ない。

なぜなら、資本主義的生産様式とその階級的抑圧機構である国家を廃棄することはなく、労働者を疎外から解放する事はなかったからだ。労働者がブルジョワ的支配から解放されるのに、権威的な介入や統制は必要だろうか？必要な所か、自発的な連帯を妨げ、むしろ有害でしかない。

よって、真の社会主義は、リバタリアン的でなければならない。さもなくば労働者が自己解放を成し遂げることは無い。20世紀の歴史でによって権威主義的社会主義の失敗はもはや目に見えている。だからこそ、ここで我々はリバタリアン社会主義を掲げる必要があるのだ。

## 運動

リバ直は、組織の名称から理解出来る様に、直接行動を我々の活動の中心に据える。以下に我々が出来ることの例を挙げるが、これだけに限定する事は無い。街宣、デモ、ビラ配り、ポスター貼り、落書き、労働運動、ストライキ、学生運動、全学バリケード封鎖、炊き出し、困窮者支援、一時的自律ゾーン、スクワッター、状況の創出、などなど…

**何より我々は、自発的そして遊戯的な革命実践を是とする**

逆に言えば、自発性に依拠しない活動は否定される。議会進出や交渉など「代表」を立てて民衆を政治から切り離す一切の権威的手法、間接行動は、反革命行為であり許されない。あくまで直接行動を至上のものとする。

また、我々が目指すべき新しい社会の仕組み、生産様式／交換様式を考察し、予示的政治(Prefigurative Politics)として行動の実践に移し、社会関係としての特徴を吟味し試行錯誤しながら、国家と資本主義を廃棄した後に開かれる地平、リバタリアン社会主義の建設を実際に進める必要がある。

## 組織

意思決定はリバタリアン的原則に則り、関係者各人の自由意志による合意を基本とする。我々に加盟する者に求められる条件は、この綱領と以下の原則への同意である。

1. 個人の自由意志と自発性の擁護
2. 近代の国家と資本主義の根本的否定
3. 新しい自律的な社会関係の創造

このリバタリアン的原則に依拠した上で、君が望む様に好きな行動を提起し、自発的な参加者を集める事ができる。**可能性は無限大だ**。なぜなら、誰も偉そうに君に命令してくるボスは居ないのだから！我がリバ直の組織そのものが、リバタリアン社会主義を体現する必要がある。

ここに、アナキストやサンディカリストから、オートノミスト、評議会共産主義者、トロツキストに至るまで、反権威主義的社会主義諸勢力の幅広い結集を呼び掛ける。

_万国の労働者よ、再結集せよ！_
